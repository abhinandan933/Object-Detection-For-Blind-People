# Object-Detection-For-Blind-People
A real-time object detection system for visually impaired people using Python, OpenCV, and YOLO. The system identifies objects and provides audio feedback in Kannada using the developer's own voice for improved accessibility and user experience.

# Project Description

Object Detection for Blind People is a real-time assistive system developed to help visually impaired individuals identify objects in their surroundings. The project uses YOLO (You Only Look Once) and OpenCV for object detection and provides audio feedback in Kannada. The audio responses are recorded in the developer's own voice, making the interaction more natural and user-friendly.

# Objectives
Assist visually impaired people in identifying objects around them.
Provide real-time object detection with audio guidance.
Improve accessibility using regional language support.
Deliver personalized voice feedback through custom-recorded audio.

# Features
Real-time object detection using YOLO.
Live camera input processing with OpenCV.
Kannada audio feedback.
Developer's own voice recordings for object announcements.
User-friendly and accessible design.
Fast and accurate object recognition.

# Technologies Used
Python
OpenCV
YOLO (You Only Look Once)
NumPy
Custom Voice Audio Files

# System Workflow
Capture live video through a camera.
Process each frame using YOLO object detection.
Identify objects present in the scene.
Trigger the corresponding Kannada audio message.
Inform the user about detected objects through voice output.

# Benefits
Enhances independence for visually impaired users.
Provides instant awareness of nearby objects.
Supports regional language communication.
Offers a personalized user experience through custom voice recordings.
